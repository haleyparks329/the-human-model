# IMU Notes

Initial sensing concept: use an IMU sensor to estimate joint angle and movement consistency.

## Possible First Experiment

Track a simple movement such as a curl, squat, or hinge.

Capture:

- Orientation
- Rep timing
- Approximate range of motion
- Tempo consistency
- Drift across repeated reps

## Success Criteria

- Sensor can stream usable data
- Data can be logged to a file
- A simple plot shows movement cycles
- At least one movement-quality metric can be extracted
