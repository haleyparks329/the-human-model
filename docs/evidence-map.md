# Public Evidence Map

The Human Model spans private working implementations and public sanitized demonstrations. This repository is the public home for the project, but it does not contain the full private automation system, private health records, private Notion data, or local configuration.

This page maps the major public claims to the evidence available here.

## Evidence Categories

- `Public runnable`: code in this repository can be run directly with mock data.
- `Public visual evidence`: screenshots, SVGs, or video assets in this repository show the behavior or interface.
- `Private implementation represented publicly`: the working implementation lives in a private or sibling repository, and this repository documents or demonstrates its shape.
- `Architecture or design only`: the repository documents the intended boundary or contract, but no live implementation is published here.
- `Experimental`: the capability exists as a prototype or early review surface and should be read narrowly.
- `Future`: planned direction, not implemented evidence.
