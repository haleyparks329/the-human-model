# Sample Output

These files are curated public outputs generated from mock data. They are small enough to review in GitHub and are intended to make the public examples easier to verify without publishing private implementation code or private records.

Fixed dates are used for determinism. These outputs are not medical advice, not autonomous coaching, and not evidence of a validated predictive model.

| Artifact | Source example | Purpose | Evidence level |
|---|---|---|---|
| [readiness-modeling-report.md](readiness-modeling-report.md) | [`examples/readiness_modeling_demo.py`](../readiness_modeling_demo.py) | Shows a readable transparent readiness report with missing-data notes and a training posture. | Public runnable |
| [bridget-daily-card.svg](bridget-daily-card.svg) | [`examples/daily_card_demo.py`](../daily_card_demo.py) | Shows the mock Bridget daily-card visual format. | Public runnable visual |
