# MediaPipe RDL Form Demo

This folder contains a screen recording of the first local MediaPipe movement-analysis prototype for Romanian deadlift review.

## Recording

- [`rdl-form-mediapipe-overlay-demo.mov`](rdl-form-mediapipe-overlay-demo.mov)

The recording shows an RDL form video with a pose overlay and movement chart. It demonstrates the early computer-vision path for turning a normal training video into reviewable movement evidence.

## What It Demonstrates

- MediaPipe pose tracking applied to an RDL video
- Visual overlay for form review
- Charted movement signal during the lift
- First step toward rep-level movement-quality analysis

## Current Scope

This is an early prototype demo. It is not a generalized exercise coach or finished form-correction system. The current value is narrower: make movement analysis inspectable by showing the tracked body position and related signal output alongside the original lift.

The private implementation continues this direction with rep-level metrics, annotated playback, angle trends, and dashboard review flags.
